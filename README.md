## セクション構成

1. **ヘッダー** — ロゴ・ナビ・CTAボタン（スマホはハンバーガーメニュー）
2. **ファーストビュー** — メインコピー・サブコピー・CTA・信頼バッジ
3. **お悩み訴求** — 6つの課題カード
4. **field flowができること** — 5サービスカード（画像付き2枚・テキスト3枚）
5. **選ばれる理由** — 4つの強みカード（SVGアイコン）
6. **制作メニュー** — 3プランカード（ライト・スタンダード・改善サポート）
7. **制作実績・ポートフォリオ** — メイン画像＋3枚のカード
8. **制作の流れ** — 6ステップグリッド
9. **料金目安** — 料金テーブル
10. **よくある質問** — 開閉式アコーディオン（5問）
11. **無料相談CTA** — 背景画像・相談ボイス・CTAボタン
12. **フッター** — ロゴ・ナビ・免責事項・コピーライト

## 実装機能

- **レスポンシブ対応** — PC / タブレット（768px〜1024px）/ スマホ（〜768px / 〜430px）
- **ハンバーガーメニュー** — スマホ時のオーバーレイメニュー（Escキー対応）
- **FAQアコーディオン** — 開閉式（1問ずつ展開）
- **スムーススクロール** — ヘッダー高さを考慮したオフセット付き
- **スクロールアニメーション** — IntersectionObserverによるfade-up
- **ヘッダーfixedスクロール** — スクロール量に応じてシャドウを変化
